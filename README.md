# Automatic-EDA
This repository provides automatic (EDA) and visualization using tools like PyGWalker, D-Tale, and ydata-profiling, enabling quick insights, interactive exploration, and automated data reports with minimal code.

It focuses on fast data understanding without writing extensive visualization or analysis logic.

---

## Tools Included

### PyGWalker

* Interactive, drag-and-drop data exploration
* Tableau-like visual analytics in Python

---

### D-Tale

* Interactive browser-based data inspection
* Supports filtering, sorting, correlations, and visualizations

---

### ydata-profiling (Pandas Profiling)

* Automated HTML reports with statistics, distributions, and warnings
* Ideal for quick dataset overview

---
